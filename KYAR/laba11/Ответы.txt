1. Sass (Syntactically Awesome Style Sheets) - это язык на основе CSS, используемый для создания более мощных и гибких стилей.
2. Для установки Sass необходимо установить Node.js, после чего можно использовать менеджер пакетов npm для установки Sass.
3. SCSS (Sassy CSS) - препроцессор CSS, используемый для упрощения создания стилей и поддержки более сложных конструкций.
4. Переменные в SCSS определяются с помощью ключевого слова $, а имя переменной может состоять из букв, цифр и нижних подчеркиваний.
5. Переменные делятся на два типа: локальные и глобальные. Локальные переменные могут быть использованы только в блоке, где они были определены, в то время как глобальные переменные могут использоваться в любом месте кода.
6. Директива @import используется для импортирования другого файла Sass в текущий документ.
7. Шаблонные селекторы - это конструкции, используемые для применения стилей к группе элементов с определенными общими характеристиками.
8. Миксины - это конструкции, используемые для повторного использования кода. Они отличаются от шаблонных селекторов тем, что они имеют возможность принимать значения параметров, что позволяет гибко применять один и тот же код к разным элементам.
9. Директива @mixin используется для создания миксина, то есть группы правил, которую можно включать в другие правила.
10. Функции в Sass создаются с помощью директивы @function.
11. Операторы Sass включают в себя: арифметические (+, -, *, / и т.д.), логические (==, !=, >, <, и т.д.), операторы сравнения (if, else, и т.д.), директивы цикла (for, each, while, и т.д.) и директивы условия (if, else и т.д.).
12. 
$text-size: 20px; 
$text-color: red;

p {
  font-size: $text-size;
  color: $text-color;
}

13. Арифметические операторы включают в себя +, -, *, / и т.д.
14. Операторы сравнения включают в себя ==, !=, >, <, и т.д.
15. Директивы цикла включают в себя @for, @each, @while, и т.д., а директивы условия включают в себя @if, @else и т.д.
16. 
$background-color: #f8f8f8;

p {
  font-size: 20px;
  color: red;
  background-color: $background-color;
  
  &:hover {
    color: #000;
    background-color: #ccc;
  }
}
